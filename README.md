# 🔥💧 Огонь и Вода — 2D платформер

Кооперативная игра для двух игроков, вдохновлённая Fireboy and Watergirl.

---

## 📦 Зависимости

```bash
pip install pygame
```

Протестировано на Python 3.9+, Pygame 2.x.

---

## 🚀 Запуск

### Одиночная игра (оба персонажа на одном ПК)

```bash
cd fire_and_water
python main.py
```

В меню выберите **«Играть (одиночно)»**.

- **Огонь** — WASD
- **Вода** — Стрелки ← → ↑

---

### Мультиплеер по локальной сети (LAN)

#### Игрок 1 — создаёт сервер

```bash
python main.py
```

1. В меню → **«Мультиплеер (LAN)»**
2. Нажмите **«Создать сервер»**
3. Сообщите ваш IP-адрес второму игроку

#### Игрок 2 — подключается

```bash
python main.py
```

1. В меню → **«Мультиплеер (LAN)»**
2. Введите IP-адрес первого игрока
3. Нажмите **«Подключиться»**

> Используемый порт: **5555 TCP**. При необходимости откройте его в брандмауэре.

## 🎮 Управление

| Действие | Огонь (WASD) | Вода (Стрелки) |
|----------|-------------|----------------|
| Влево    | A           | ←              |
| Вправо   | D           | →              |
| Прыжок   | W           | ↑              |
| Меню     | ESC         | ESC            |
| Перезапуск | R         | R              |

## 🔥💧 Правила

- **Огонь** гибнет в лужах воды (синие ловушки)
- **Вода** гибнет в лаве (красные ловушки)
- Персонажи **не могут касаться друг друга**
- Нажимайте кнопки вместе, чтобы открывать двери
- Оба должны достичь своих выходов (🔥 и 💧)
